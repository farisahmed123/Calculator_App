## üöÄ Getting Started

Follow these steps to run the app on your local machine.

### Prerequisites
*   Node.js installed on your system.
*   Git installed.

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/farisahmed123/Calculator_App.git
    cd Calculator_App
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run the App**
    ```bash
    npm start
    ```

4.  **Build .exe (Windows)**
    ```bash
    npm run dist
    ```
